## Abstract

Qwen2.5-Max is a large-scale multimodal language model demonstrating state-of-the-art performance on mathematical reasoning, code generation, and multilingual tasks.

## Key Results

- **MATH**: 85%+ accuracy
- **GSM8K**: 98%+ accuracy
- **HumanEval**: 90%+ code generation
- **Multilingual**: Strong across 29 languages

## Architecture

- **Mixture of Experts**: Efficient parameter utilization
- **Multimodal**: Native image and video understanding
- **Long context**: 128K+ token support
- **Agentic**: Tool use and autonomous planning

## Training

- **Data**: High-quality mathematical and coding corpora
- **RLHF**: Human preference alignment
- **Distillation**: Efficient smaller variants

Core models and weights available on Hugging Face.
