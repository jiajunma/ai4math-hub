## Abstract

We present a generalist neural theorem prover trained on multiple proof assistants including Lean, Coq, and Isabelle. The same model can generate proofs across different formal systems.

## Key Contributions

- **Multi-system**: One model for Lean, Coq, Isabelle
- **Cross-domain transfer**: Learns patterns across systems
- **Strong results**: Competitive with specialist models
- **Unified approach**: Reduces need for system-specific models

## Significance

Shows that neural theorem proving can generalize across formal systems.
