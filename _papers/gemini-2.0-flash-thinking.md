## Abstract

Gemini 2.0 Flash Thinking is an experimental model featuring enhanced reasoning capabilities with native multimodal understanding. The model generates implicit chain-of-thought for complex problems.

## Key Features

- **Multimodal reasoning**: Text, images, video, audio
- **Implicit CoT**: Internal reasoning without explicit tokens
- **Speed**: Optimized for fast inference
- **Context**: 1M+ token context window

## Performance

- **Math benchmarks**: Competitive with top models
- **Visual reasoning**: Strong geometry and diagram understanding
- **Scientific reasoning**: Physics, chemistry, biology tasks

## Applications

- Mathematical problem solving
- Scientific research assistance
- Educational tutoring
- Code generation with reasoning

## Availability

Available through Google AI Studio and Vertex AI.
