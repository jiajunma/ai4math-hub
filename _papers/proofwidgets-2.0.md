## About

ProofWidgets 2.0 is a major update to the interactive proof visualization system for Lean 4. It enables custom interactive widgets within the Lean infoview.

## New Features

- **Enhanced visualization**: Better rendering of proof states
- **Interactive diagrams**: Clickable proof trees
- **Custom widgets**: User-defined visualization components
- **Performance**: 3x faster rendering

## Applications

- **Education**: Visual proof exploration for students
- **Research**: Understanding complex proofs
- **Debugging**: Finding errors in proofs
- **Presentation**: Demonstrating proof techniques

## Integration

- VS Code extension support
- Web-based proof exploration
- Export to various formats

## Community

Growing ecosystem of custom widgets created by the Lean community.
