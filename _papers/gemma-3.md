## Release

Google released Gemma 3 in March 2026, the latest version of their open-weight language models.

## Key Features

- **Multiple sizes**: 1B, 4B, 12B, and 27B parameters
- **Long context**: 128K token context window
- **Multilingual**: Support for 140+ languages
- **Math reasoning**: Significant improvements on MATH and GSM8K benchmarks

## Performance

- **MATH benchmark**: Competitive with larger proprietary models
- **Code generation**: Strong performance on coding tasks
- **Instruction following**: Improved alignment and helpfulness

## Open Source

- Open weights available on Hugging Face and Kaggle
- Responsible AI license
- Research and commercial use permitted

## Significance

Gemma 3 provides state-of-the-art open models that researchers and developers can use for mathematical AI applications.
