## Abstract

We introduce FunSearch, a method to search for new solutions in mathematics and computer science using Large Language Models. FunSearch combines a pre-trained LLM with an automated evaluator to search for solutions in the space of computer programs.

## Key Contributions

- **First LLM mathematical discovery**: Published in Nature
- **Cap set problem**: New bounds on the cap set problem in combinatorics
- **Bin packing**: Improved online bin packing algorithms
- **Program search**: Searches in the space of programs rather than direct answers

## Method

1. **LLM** generates program variations
2. **Evaluator** scores each program
3. **Evolution** keeps best programs
4. **Iteration** continues until improvement plateaus

## Significance

First demonstration that LLMs can contribute novel mathematical knowledge, not just reproduce existing results.
