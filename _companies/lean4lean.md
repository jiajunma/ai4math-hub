## About

The lean4lean project aims to formally verify the Lean 4 theorem prover using Lean 4 itself. This creates a self-hosting verified system where the kernel is proven correct.

## Goals

1. **Verified Kernel**: Prove correctness of Lean's typechecker
2. **Self-hosting**: Use Lean to verify Lean
3. **Trust**: Increase confidence in formal proofs checked by Lean

## Significance

This is a major milestone in formal verification, creating one of the first self-verifying theorem provers.
