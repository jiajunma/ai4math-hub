## About

SageMath is a free, open-source mathematics software system. It combines many specialized mathematics software packages into a single Python-based interface.

## Features

- **Algebra**: Groups, rings, fields
- **Calculus**: Symbolic and numerical
- **Number theory**: Elliptic curves, modular forms
- **Geometry**: 2D/3D plotting, polytopes
- **Statistics**: Comprehensive statistics

## Components

SageMath wraps and integrates:
- NumPy, SciPy
- SymPy
- Matplotlib
- PARI/GP
- GAP
- Singular
- And many more
