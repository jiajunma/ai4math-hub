## About

Cohere is an AI company focused on building enterprise-grade language models. Their Command series models demonstrate strong capabilities in reasoning tasks.

## Products

- **Command R+**: Advanced reasoning model
- **Embed**: Embedding models for search
- **Rerank**: Search relevance ranking
- **Fine-tuning**: Custom model training

## Math Capabilities

Command R+ shows competitive performance on:
- Mathematical reasoning benchmarks
- Code generation and analysis
- Scientific text understanding

## Enterprise Focus

- Data privacy and security
- On-premise deployment
- Custom model training
- API and cloud options

## Team

Founded by former Google Brain researchers including Aidan Gomez (co-author of "Attention is All You Need").
