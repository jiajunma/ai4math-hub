## About

PVS is a verification system developed at SRI International. It combines a specification language with a powerful theorem prover.

## Applications

- **Aerospace**: NASA, FAA certification
- **Hardware verification**: Processor designs
- **Protocol verification**: Security protocols

## Features

- **Rich type system**: Predicate subtypes, dependent types
- **Decision procedures**: Integrated SMT solving
- **Proof strategies**: User-defined tactics
- **Model checking**: Integration with model checkers

## Industrial Impact

Used in certification of:
- Flight control systems
- Air traffic control
- Medical devices
