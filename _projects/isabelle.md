## About

Isabelle is a generic proof assistant developed at Cambridge University and TU Munich. It allows mathematical formulas to be expressed in a formal language and provides tools for proving those formulas.

## Features

- **Higher-Order Logic**: Expressive formal language
- **Isar**: Human-readable structured proof language
- **Sledgehammer**: Automated proof search
- **Archive**: AFP (Archive of Formal Proofs)

## Applications

- Formal verification of seL4 OS kernel
- Formalized mathematics (e.g., Prime Number Theorem)
- Verification of cryptographic protocols

## Ecosystem

- **Isabelle/HOL**: Main logic
- **Isabelle/ZF**: Zermelo-Fraenkel set theory
- **AFP**: Community-contributed formalizations
