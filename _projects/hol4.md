## About

HOL4 is an interactive theorem prover for higher-order logic. It is used extensively in industrial verification, particularly at ARM.

## Industrial Use

- **ARM Processor Verification**: Used to verify ARM processors
- **Security**: Verified security properties
- **Compiler Verification**: CakeML verified compiler

## Features

- **Higher-order logic**: Expressive formalism
- **Tactics**: Powerful proof automation
- **Proof recording**: Complete proof reconstruction
- **Embeddings**: Deep and shallow embeddings of languages

## Notable Projects

- **CakeML**: Verified ML compiler
- **seL4**: OS kernel verification (partial)
- **ARM models**: Processor architecture specifications
