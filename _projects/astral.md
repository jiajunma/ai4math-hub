## About

Astral is a neural network-based proof assistant for Lean 4. It provides intelligent tactic suggestions and can search for complete proofs automatically.

## Features

- **Tactic prediction**: Suggests next tactics based on proof state
- **Proof search**: Complete proof automation
- **Neural backend**: Transformer-based models
- **Lean 4 integration**: Native support for latest Lean

## Technology

- Transformer models trained on mathlib4 proofs
- Beam search for proof exploration
- Reinforcement learning from proof attempts

## Status

Active development by Lean community members.
