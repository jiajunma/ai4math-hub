## About

Metamath is a language and a proof checker for developing and verifying strictly formalized mathematical definitions and proofs. It's notable for its extreme simplicity.

## Features

- **Simple Core**: Only a few hundred lines of code
- **Verifiable**: Small trusted kernel
- **Expressive**: Can encode ZFC set theory
- **Database**: Large library of formalized math

## Metamath Proof Explorer

A web-based interface to the Metamath database with over 23,000 proofs.

## Applications

- Foundation verification
- Teaching formal methods
- Minimalist proof checking
