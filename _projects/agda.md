## About

Agda is a dependently typed functional programming language and proof assistant. It is used for formal verification and constructive mathematics.

## Features

- **Dependent types**: Full dependent type system
- **Proof automation**: Tactics and reflection
- **Haskell-like**: Familiar syntax
- **Emacs/VS Code**: Editor integration

## Applications

- Formal verification of programs
- Constructive mathematics
- Type theory research
- Category theory formalization

## Cubical Agda

Extension for homotopy type theory:
- Univalence axiom
- Higher inductive types
- Cubical type theory

## Comparison

- Similar to: Coq, Lean, Idris
- Different: More focus on dependently typed programming
- Community: Smaller but active
