## About

Lean 4 is the latest version of the Lean theorem prover, designed to be both a powerful tool for formalized mathematics and a practical programming language.

## Key Improvements over Lean 3

- **Performance**: Significantly faster compilation and execution
- **Metaprogramming**: Better support for writing tactics and macros
- **Interoperability**: Easier to call external code
- **Compiler**: Produces efficient native code

## Ecosystem

- **mathlib4**: Comprehensive mathematics library
- **Lake**: Modern package manager
- **Lean LSP**: Language server protocol support
- **VS Code extension**: Excellent IDE integration

## Adoption

Used by:
- **Xena Project**: Formalizing undergraduate mathematics
- **LeanDojo**: Neural theorem proving research
- **Software verification**: Verified systems software
- **Teaching**: Multiple university courses
