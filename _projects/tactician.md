## About

Tactician is a machine learning-based system for suggesting tactics in Coq proofs. It learns from existing proofs to predict the next tactic.

## Features

- **Real-time suggestions**: As you type in Coq
- **Learning**: Adapts to your proof style
- **Integration**: Works with Proof General and CoqIDE
- **Neural**: Uses graph neural networks

## How it works

1. Analyzes the current proof state
2. Predicts likely next tactics
3. Ranks by probability
4. User selects or continues

## Impact

One of the first practical AI assistants for interactive theorem proving.
