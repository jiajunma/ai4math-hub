## About

Dafny is a verification-aware programming language. It includes built-in specification constructs and verifies programs using Z3 SMT solver.

## Features

- **Pre/post conditions**: Hoare logic style
- **Loop invariants**: Automated verification
- **Ghost code**: Specification-only code
- **Inductive datatypes**: Algebraic data types

## Verification

Uses Microsoft Z3 SMT solver for:
- Program verification
- Type checking
- Proof obligations

## Educational Use

Widely used in teaching formal methods:
- University courses
- Online tutorials
- Verification competitions

## Microsoft Research

Developed at Microsoft Research with active maintenance.

- Algorithm verification
