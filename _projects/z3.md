## About

Z3 is a state-of-the-art SMT (Satisfiability Modulo Theories) solver from Microsoft Research. It is one of the most widely used automated reasoning tools in both academia and industry.

## Capabilities

- **SMT solving**: Combines SAT solving with theory reasoning
- **Theories**: Arithmetic, arrays, bit-vectors, uninterpreted functions
- **Optimization**: MaxSAT and optimization modulo theories
- **Quantifiers**: E-matching and MBQI for quantified formulas

## Applications

- **Software verification**: Used in tools like Dafny, Boogie
- **Symbolic execution**: KLEE, PEX
- **Test generation**: Whitebox fuzzing
- **Program synthesis**: SKETCH, Rosette
- **Security**: Vulnerability detection
