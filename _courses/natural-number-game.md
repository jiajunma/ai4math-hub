## Course Introduction

Natural Number Game is an interactive tutorial for learning Lean 4. Through a series of carefully designed levels, learners will master Lean's basic concepts and proof techniques while proving properties of natural numbers.

Game premise: Starting from zero, you'll prove increasingly complex theorems about natural numbers, gradually unlocking new mathematical concepts and Lean features.

## Course Content

### Worlds

1. **Tutorial World** - Basic operations
   - `rw` (rewrite) tactic
   - `rfl` (reflexivity) tactic

2. **Addition World** - Addition
   - Prove addition commutativity and associativity
   - Use `induction`

3. **Multiplication World** - Multiplication
   - Prove multiplication properties
   - Combine addition and multiplication

4. **Power World** - Exponentiation
   - Definition and properties of powers

5. **Function World** - Functions
   - Basic function concepts
   - `intro` tactic

6. **Implication World** - Implication
   - Proving logical implications

7. **Advanced Multiplication World** - Advanced multiplication
   - More complex multiplication theorems

## Learning Method

### Game Mechanics

- 🎯 Each level has a target theorem to prove
- 💡 Use hints if stuck
- ✅ Automatically advance to next level after correct proof
- 🏆 Complete a world to unlock the next

### Learning Tips

1. **Complete in order**: The tutorial has progressive design
2. **Try first**: Before looking at hints or answers
3. **Understand each step**: Don't just copy-paste
4. **Use community**: Ask questions on Lean Zulip
